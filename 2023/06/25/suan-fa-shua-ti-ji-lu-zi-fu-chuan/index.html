    151.翻转字符串里的单词

    关键代码：

    去掉多余的空格

    void removeExtraSpaces(string& s)
{
    int slow=0;
    for(int i=0;i<s.size();++i)
    {
        if(s[i]!=' ')
        {
            if(slow!=0) s[slow++]=' ';
            while(i<s.size() && s[i]!=' ')
                s[slow++] = s[i++];
        }
    }
    s.resize(slow);
}

    剑指offer 58:左旋转字符串

    string reverseLeftWords(string s,int n)
{
    reverse(s.begin(), s.begin()+n);
    reverse(s.begin()+n,s.end());
    reverse(s.begin(),s.end());
    return s;
}

    KMP算法

    前缀表（prefix table）：记录了模式串与主串（文本串不匹配的时候，模式串应该从哪里开始匹配）

    记录下标i之前（包括i）的字符串中，有多大长度的相同前缀后缀

    最长公共前后缀

    前缀是指不包含最后一个字符的所有以第一个字符开头的连续子串

    后缀是指不包含第一个字符的所有以最后一个字符结尾的连续子串

    前缀表要求的就是相同前后缀的长度

    使用next数组来匹配

    如果s[i] s[j+1]不相同，也就是 前后缀末尾不相同的情况，要向前回退。

    459.重复的字符串

    移动匹配

    如果字符串由重复的子串组成，则s+s中必定出现一个s

    其中需要用到find库函数

    下面是使用方法

    class Solution { 
public:     
    bool repeatedSubstringPattern(string s) {         
        string t=s+s;         
        t.erase(t.begin());         
        t.erase(t.end()-1);                        if(t.find(s)!=std::string::npos)         
            return true;          
        return false;     
    } };

    KMP解法

    当一个字符串由重复子串组成，最长相等前后缀不包含的字串就是最小重复子串

    假设字符串s使用多个重复字串组成，重复出现的子字符串长度是x

    所以s由n*x组成

    因为字符串的最长相同前后缀长度一定是不包含s本身

    所以最长相同前后缀长度必然是m*x

    且n-m=1

    如果nx%(n-m)x=0

    就可以判定由重复出现的字符串

    如果len%(len-(next[len-1]+1))==0

    说明数组长度可以被（数组长度-最长相等前后缀长度）整除

    数组长度减去最长相同前后缀的长度相当于是一个周期的长度
