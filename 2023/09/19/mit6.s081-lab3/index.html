    Lab: page tables

    在本实验中，您将探索页表并修改它们以简化将数据从用户空间复制到内核空间的功能。

    在开始编码之前，请阅读xv6书的第3章以及相关文件：

    • kern/memlayout.h，它描述了内存的布局。
    • kern/vm.c，其中包含大部分虚拟内存（VM）代码。
    • kernel/kalloc.c，其中包含用于分配和释放物理内存的代码。

    切换到pgtbl分支来开始实验

    $ git fetch
$ git checkout pgtbl
$ make clean
