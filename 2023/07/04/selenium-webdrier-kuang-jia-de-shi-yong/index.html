Selenium-Webdrier框架的使用

Webdriver

面向对象的API

有效的驱动浏览器

操作浏览器的基本方法

打开网页

#导入Webdriver模块
from selenium import webdriver
#打开浏览器
driver = webdriver.Chrome() #谷歌
#driver = webdriver.Firefox() 火狐
#driver = webdriver.Ie() IE
#打开网站
driver.get("http://www.baidu.com")

设置休眠

由于打开网址后，页面加载需要几秒钟，定时等待几秒钟等到页面加载完毕后再进行下一步操作

from selenium import webdriver
import time
driver = webdriver.Chorme()
driver.get("http://ww.baidu.com")
time.sleep(3)

页面刷新

driver.refresh()

页面切换

#返回上一页
driver.back()
time.sleep(3)
#切换到下一页
driver.forward()
time.sleep(3)

设置窗口大小

#设置窗口大小
driver.set_window_size(1920, 1080)
time.sleep(2)
#将浏览器窗口最大化
driver.maximxze_window()

截屏

截屏后可以设置保存路径+文件名+后缀

driver.get_screenshot_as_file("D//screenshot.jpg")

退出

退出有两种方式

close：关闭当前窗口

quit：结束进程，关闭所有窗口

结束测试用quit，可以回收c盘的临时文件

driver.close()
driver.quit()

请求浏览器信息

包括窗口句柄、浏览器尺寸位置、cookie

建立等待策略

在尝试定位元素之前，确保该元素位于页面上，并且在尝试与该元素交互之前，该元素处于可交互状态

driver.implicitly_wait(0.5)

发送命令、查找元素

在Selenium会话中，要先找到元素才能与其交互

text_box = driver.find_element(by=By.NAME, value="my-text")
submit_button = driver.find_element(by=By.CSS_SELECTOR, value="button")

操作元素

text_box.send_keys("Selenuium")
submit_button.click()

获取元素信息

value = message.text
