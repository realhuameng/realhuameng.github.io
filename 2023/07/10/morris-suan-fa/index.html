    Morris算法可以将二叉树遍历的空间复杂度降到O(1)

    Morris遍历的实现

    Morris遍历的规则

    当前节点为cur

    开始时cur指向一整棵树的头节点

    -cur无左子树

    cur向右移动（cur=cur.right）

    -cur有左子树

    找到cur左子树上最右的节点，记为mostright

    如果mostright的right指针指向NULL，让其指向cur（mostright.right=cur），cur向左移动（cur=cur.left）

    如果mostright的right指针指向cur，让其指向NULL（mostright.right=NULL），cur向右移动（cur=cur.right）

    没有左子树的节点只到达一次，有左子树的节点到达两次

    public static void morris(Node head){
    if(head == null) return;
    Node cur = head;
    Node mostright = NULL;
    while(cur != NULL)
    {
        mostright = cur.left;
        if(mostright != NULL) //有左树
        {
            while(mostright.right != NULL && mostright.right != cur)
                mostright = mostright.right; //找到最右的节点
            if(mostright.right == NULL)
            {
                mostright.right = cur;
                cur = cur.left;
                continue;
            }
            else
            {
                mostright.right = NULL;
            }
            
        }
        cur = cur.right;
    }
}

    先序遍历：第一次到达就打印

    中序遍历：只能到达一次的节点直接打印，能到达第二次的节点第二次打印

    后序遍历：第二次到达节点的时候打印节点左子树的逆序右边界
